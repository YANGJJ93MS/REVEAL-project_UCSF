The executable for the multi-threaded linux version of MS-DIAL is located at Venodo. 
https://zenodo.org/records/14736627
Alternatively, MS-DIAL can be downloaded from its official site at 
https://systemsomicslab.github.io/compms/msdial/main.html. This site also contain
executables that run on Windows and Macs.

To process the data type the following command:

$dir_containing_executable/MsdialConsoleApp lcmsdda -i $data_dir -o $data_dir -m $data_dir/POS-Msdial-lcms-dda-Param.txt

the output (-o) can be directed into another directory is preferred.
